In this exercise we have components like app-Message-Panel, app-Show-Message and app-Clear-Message.

On Click of Show Message

  1. Display "Some Message"
  2. Disable Show Message Button
  3. Change the color from Blue to grey
  4. Enable Clear Message Button & change Color to Blue

On Click of Clear Message

  1. Clear the Message displayed in previous step
  2. Disable Clear Message Button
  3. Change the color from Blue to grey
  4. Enable Show Message Button & change Color to Blue
